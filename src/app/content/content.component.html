Welcome to Northumberland

We have developed an Android app to aid tourists and architectural enthusiasts easily find places to visit in the Northumbria area. The app shows the location of buildings in relation to the users location and allows the ability to learn about these buildings in the app aswell as get directions to them.

The Map

The map screen is the main page of the app and is similar to any other map application allowing it to feel familiar and simple to use. You can tap on a pin to view the name of it, a brief summary and an image of the location, furthermore you can get directions to the location (this will open google maps externally).

Searching for a Location

The intuitive search screen allows for a user to easily view information about nearby locations and the ditance to them. If the user only wants to view locations up to a certain distance from them then they can use the slider at the top to be able to do this. The checkboxes at the top allow for further refinement of the search, such as only showing places that are child or wheelchair friendly. Finally the search bar allows a user to search for a place or by location type (castle, theatre, etc).

Location Timeline

The timeline screen allows easy viewing of locations by date, allowing a user to see which are the oldest or most recent and depending on what you want to view it is easy to change what is shown with the 2 dropdown menus at the top

Settings

By storing users preferences in a settings file, we allow for consistent use of the app each time it is opened and for ease of use using features such as editing the font size, colour or text to speech.
